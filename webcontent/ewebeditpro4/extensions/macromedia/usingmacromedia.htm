Using Dreamweaver and UltraDev to Insert eWebEditPro into Web Pages

Using the Extension to Insert eWebEditPro into a Web Page Form

In this procedure, you create a form and then use the extension to insert eWebEditPro into the form.

1. Open Dreamweaver.

2. Display HTML source code. To do this:

  •  in Dreamweaver 3, select Window>HTML source            
  •  in Dreamweaver 4 Ultradev, click the Show Code View icon ()

3. Make sure that the cursor appears between the body tags.

4. To insert a form, select Insert>Form from the verb strip menu.

5. Place the cursor between the Form tags.

6. Click Commands > Ektron eWebEditPro from the verb strip menu.

7. A dialog box appears. Enter values into the fields to determine key information about the editor.

In the top section of the dialog box, you specify the:

  • Editor Name            
  • Width         
  • Height         
  • Content or Content as a variable name

The following table explains these fields. The lower section of the dialog box, SETUP, is explained below.

 Field

 Enter

 Editor Name

A name to assign to this instance of eWebEditPro.

The name must be a valid JavaScript identifier and follow these guidelines.

• It consists of only ASCII letters and digits, underscores (_) and dollar signs ($).

• The first character cannot be a digit.

• Spaces are not permitted

JavaScript you enter to customize eWebEditPro typically refers to the editor name.

 Width

The width of the editor on the page.

Enter the size in pixels or as percentage of the screen size. To distinguish a percentage, follow the number with a percent sign (%).

The drop down list includes some common sizes to choose from.

 Height

The height of the editor on the page.

Enter the size in pixels or as percentage of the screen size. To distinguish a percentage, follow the number with a percent sign (%).

The drop down list includes some common sizes to choose from.

 Content

Initial content to be inserted into the editor when the user first opens this Web page.

 Content as a variable name

 

If you cannot fit the initial content into the Content field (above), enter the text in a variable defined   elsewhere in the file. Then, enter the variable name in this field.

Note:  This field is inactive in Dreamweaver 3.

Setup Section of the eWebEditPro Dialog Box

The lower section of the eWebEditPro dialog box lets you specify the location of the ewebeditpro directory and your dynamic web server software. The following table explains these fields.

 Field

 Description

eWebEditPro Location

Enter the directory to which eWebEditPro is installed, relative to the web root directory. For example,  /ewebeditpro/.

This directory is set in the ewebeditpro.js file.
Server

Click the dynamic web server software you are using. 

Update Setup

When the Setup information is correct, click this button to save it. When you do, you insert eWebEditPro into your Dreamweaver template.

Help

If your PC has an internet connection, pressing this button displays this page on the Ektron web site.

8.  When you complete the dialog box, click Insert. When you do, the code to invoke the eWebEditPro editor is inserted into the Web page and assigned to an image in Dreamweaver. The image (illustrated below) indicates where you placed the editor. (In Dreamweaver 4, switch to Design View to see the image.)
