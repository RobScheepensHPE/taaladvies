Using Dreamweaver and UltraDev to Insert eWebEditPro into Web Pages

Prerequisites

The following software must be installed on your PC.

  • Ektron eWebEditPro server software, version 2.0.0.34 or higher                 
  • Macromedia Dreamweaver 3, Dreamweaver 4, or Dreamweaver UltraDev 4

Note: If you are running Dreamweaver 3, you must first install the Macromedia Extension Manager, available from the  Macromedia Website at http://www.macromedia.com/exchange/dreamweaver/.

  • Dynamic web server software, one of the following: ASP, ColdFusion, JRUN, or JSP

Adding the eWebEditPro Extension into Dreamweaver

In this procedure, you install an extension file that lets you insert eWebEditPro into a web page form.

  • If you downloaded the ewebeditpromacromedia.zip file, unzip it in the directory on the server where eWebEditPro was installed. For example, C:\Inetpub\wwwroot\ewebeditpro2. Be sure to preserve the subdirectories of the ZIP file. This creates the directory:

...\ewebeditpro2\extensions\macromedia 

(eWebEditPro is typically installed to the C:\Inetpub\wwwroot\ewebeditpro2 directory.  However, it may have been installed to a different directory.)

1. Open Dreamweaver.

2. From the verb strip menu, click Commands > Manage Extensions.

3. The Macromedia Extension Manager window appears. From the verb strip menu, click File>Install Extension.

4. The Select Extension to Install window appears. Browse to the folder on your PC that contains the Macromedia Extension Package (.mxp) files. When you install, these files are placed in the ewebeditpro2\extensions\macromedia  directory.

(eWebEditPro is typically installed to the C:\Inetpub\wwwroot\ewebeditpro2 directory.  However, it may have been installed to a different directory.)

5. Select  ewebeditpro.mxp.

6. Click the Install  button and follow the screen instructions. 

7. When you finish, save everything then exit the program. 

8. Restart the application. 

You have installed the eWebEditPro extensions. Now, you can begin inserting eWebEditPro into a Dreamweaver form.
