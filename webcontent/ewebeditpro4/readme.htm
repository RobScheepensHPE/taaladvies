Thank you for your interest in eWebEditPro.

Please take the time to familiarize yourself with the integration documentation for your server platform. It contains all the information you need to integrate and use eWebEditPro in your web-based application.

The eWebEditPro ActiveX control can only be used in web pages. It cannot be used as part of another application, such as one developed with Visual Basic.

Certain keyboard keys do not function in Netscape as they do in Internet Explorer. The following keys are among those that are different: Delete, Ctrl-x, Ctrl-c, Ctrl-v, Ctrl-b, Ctrl-i, Ctrl-u.

Norton AntiVirus 2001 may display a warning message when using the spell checker in eWebEditPro. To enable spell checking, open Norton AntiVirus 2001, select Options, click Script Blocking and uncheck Enable Script Blocking.
