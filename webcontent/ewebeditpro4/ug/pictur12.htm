Inserting Images

Editing the Picture's Title

The title that you entered in the Insert Media Item dialog box defaults into the Title field of the Media Selection dialog box. You can edit the title in this field if desired.

Note

The title is also the alt text for the image. The alt text appears in place of the image on the Web page if the image itself cannot display for any reason.
