Checking Spelling

Disabling Script Blocking

If Norton Antivirus TM 2001 is installed on your computer, you need to disable script blocking in order to use the spell checker. If you do not disable script blocking, an error message will appear whenever you check spelling.

To do this, follow these steps.

  1. Launch Norton Antivirus TM 2001.
  2. From the first window, click Options.
  3. On the next window, click Script Blocking.
  4. On the script blocking window, uncheck Enable Script Blocking.
  5. Press OK.
