Inserting Images

Inserting a Picture from the Server

The top left corner of the dialog box (Illustrated below) lists the pictures on the Web server.

Pictures that you previously inserted appear on this list, as do pictures inserted by other users connected to that server.

To insert a picture from the server, follow these steps.

  1. Click the picture from the To Select an Existing File field that you want to insert.
  2. Information about the picture's file size, width and height appears in the upper right corner of the dialog box.
  3. If you want to view a picture before inserting it, click Preview (on the right side of the dialog box).
  4. Click OK.
  5. The Picture Properties dialog box reappears. Here, you can change the properties of the picture. For more information, see
  • Adjusting a Picture
  • Setting a Border
  • Aligning the Picture
  • Adding Space around the Picture
  1. Click OK to insert the picture into the editor.
Deleting a Picture from the Server

To delete a picture from the server, select the picture and click the Delete button.
