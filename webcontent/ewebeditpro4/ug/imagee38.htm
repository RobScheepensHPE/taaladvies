Undo

Standard Toolbar Button
Description

Reverses the most recent commands within the current editing session.

You can undo several commands at once by pressing the undo command repeatedly. The first time you press the command, the most recently completed command is "undone." The next time, the second most recently completed command is "undone", etc.

See Also: Redo
