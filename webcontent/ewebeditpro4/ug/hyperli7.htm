Using Hyperlinks

Editing a Hyperlink

If you need to change a hyperlink's destination Web page or target frame, follow these steps.

  1. Click the Hyperlink button ( ). The hyperlink dialog box appears.
  2. Edit the Link or Target Frame field as needed.
  3. Press OK.
