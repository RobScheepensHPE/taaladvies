Editing in Microsoft Word

Uploading an Image in a Microsoft Word Document

If you insert an image into the Word content and then paste that content or save it, the following dialog box appears. The box lists all images in the content and asks if you want to copy them from your computer to your organization's Web server.

If you are ready to publish your Web content and want to display the image, click Update Now. If you plan to add more content later, you can click Upload Later and upload all images at that time.
