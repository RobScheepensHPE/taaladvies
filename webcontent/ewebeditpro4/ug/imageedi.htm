Editing Images

WebImageFX lets you select an image from the content, edit that image, and then insert the updated image into the content. Editing consists of several functions available on the toolbar, such as

  • brightening
  • rotating
  • changing the color depth
  • inserting text

This chapter consists of the following main topics:

  • Using WebImageFX
  • Toolbar Buttons and Menu Commands
  • Specifying Color Depth

Important!

WebImageFX is an external, add-on product available from Ektron.
