Creating a Simple Web Page

Creating the Sample Web Page

To create this page, follow these steps.

Enter the First Two Sentences
  1. Open eWebEditPro. (Your Webmaster installs eWebEditPro on your computer and determines which fonts and sizes are available. Your system may not match the example below.)
  2. Type the first two sentences.

This is some sample content. You can make text bold, italic and underlined with the push of a button.

  1. Double click the word "bold" to select it. Then, click the Bold button ( ) to apply bold to the word.
  2. Double click the word "italic." Then, click the Italic button ( ) to apply italic to the word.
  3. Double click the word "underlined." Then, click the underline button ( ) to underline the word.
Creating a Hyperlink
  1. Type You can also easily add hyperlinks.
  2. Double click the word hyperlinks so that it becomes selected.
  3. Click the Hyperlink button ( ).
  4. The hyperlink dialog box appears.

  1. In the Link field, after http://, enter www.ektron.com.
  2. Click OK.

Notice that the word hyperlink now appears in a different color. When you save this Web page and a user views it, if the user clicks hyperlink, a new Web page will display Ektron's home page.

Note

If your computer has an internet connection, you can double click the hyperlink to test it.

Congratulations! You have just created your first Web page with eWebEditPro. You have learned how to apply bold, italic, and underlining, create a hyperlink, add a bullet to a line, and change the size of the text.
