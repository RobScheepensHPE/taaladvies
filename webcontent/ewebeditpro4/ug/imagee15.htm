Description

Lest you change an image's width and/or height, which are defined in pixels.

Maintaining Aspect Ratio

Use this check box to change an image's size while maintaining its aspect ratio.

Aspect ratio is the ratio of an image's width to height. For example, if a graphic has an aspect ratio of 2:1, its width is twice as large as its height.

If you check the Maintain Aspect Ratio box, just specify a new width -- the height is calculated automatically.
