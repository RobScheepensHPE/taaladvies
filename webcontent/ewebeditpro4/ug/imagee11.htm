Editing Images

Copy

Standard Toolbar Button

Description

Copies a selected area of an image. After you copy an image, you can paste it using the paste command. See Also: Paste as New Image

To select an area of an image, use the Select command. See Also: Select

If you do not select an area, the entire image is copied.
