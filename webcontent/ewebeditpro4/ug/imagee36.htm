Editing Images

Twain Acquire

Standard Toolbar Button

Description

Perform a single page scan from a previously selected source.

See Also: Twain Source

Importing Scanned Images

WebImageFX lets you acquire images from a scanner or digital camera that supports the Twain standard. You select the source (using Twain Source) and do a quick acquire from that source (using Twain Acquire).

Limitations

  • You can only acquire one page at a time
  • Few digital cameras support twain. Usually, digital cameras resemble a hard drive or use proprietary software to extract images. Ektron does not support an interface to proprietary software.
  • Not all scanners support the twain standard
