Working with Table Cells

Splitting a Cell

You can divide a cell into two. If you split a cell, each cell occupies one half the size of the original cell.

Row before split

Row after split

To split a table cell into two cells, follow these steps.

  1. Place the cursor in the cell that you want to split.
  2. Right click the mouse.
  3. Click Split Cell from the menu.

  1. Two cells now appear where only one appeared before.
