Editing Images

Create New

Standard Toolbar Button

Description

Lets you create a new image. If you are editing an image when you press this command, you are asked if you want to save changes to it first.

By default, a new image's bit depth is 24, but you can change it if desired using the color depth command.

See Also: Color Depth
