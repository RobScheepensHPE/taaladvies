Inserting Images

Moving an Image to the Server

If you insert an image from your computer into eWebEditPro content, that image must be moved to the server when you save the content. The image can only appear on Web pages that display it after you move it to the server .

When you save content that incudes an image you inserted, the following dialog appears.

All inserted files appear in the dialog, which also estimates how long it will take to move the files to your server. If you do not want to wait that long to upload the files, click Upload Later. If you do this, the images will not appear on the published Web content until you move them to the server.

You can also enter a Title for the image near the bottom of the screen. If you do, this text appears when the user reading the Web pages moves the cursor over the image.
