Editing Images

Line

Standard Toolbar Button

Description

Draw a straight line. To change a line's size or color, right click it after drawing it.

To use this command, click the line command, drag the line, and release the mouse. To change the line's size or color, place the cursor on the line and press the right mouse button to display the Attributes menu.
