Editing Images

Toolbar Buttons and Menu Commands

The following tables list each toolbar button and menu command. They are followed by a more detailed description of each function.

File Menu Options

Toolbar Button
Command
Brief Description
For more information, see
Create New
Creates a new image.
Create New
Open
Selects an image to edit.
Open
Save
Saves changes to an image.
Save
Save As
Saves the current image under a different name or format.
Save As
Twain Acquire
Performs a single page scan.
Before scanning, you must select a source using the Twain Source command.
Twain Acquire
Twain Source
Allows the user to select a source for acquiring an image, such as a scanner or digital camera.
Twain Source
Exit
Close WebImageFX. Copy image back to eWebEditPro.
Exit
Thumbnail
Creates a thumbnail of the current image.

Note: A thumbnail is a miniature display of an image. Thumbnails allow fast browsing though images.

Thumbnail

Edit Menu Options

Toolbar Button
Command
Brief Description
For more information, see
copy
Copies a selected area of an image.
Copy
paste as new image
Inserts copied image or area into a new file.
Paste as New Image
undo
Reverses the most recent action.
Undo
redo
Reapplies the changes from the command that occurred before you pressed Undo.
Redo
select
Selects an area of an image. You can then perform actions on the area, such as blur and delete.
Select
crop
Removes everything outside of selected area of an image.
Crop
delete
Deletes selected area of an image.
Delete
