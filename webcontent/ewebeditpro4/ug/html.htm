Working with HTML

eWebEditPro creates pages for display on the World Wide Web or an intranet. These pages use HTML (hypertext markup language) to format text and images for display in a browser. You do not need to know HTML to use eWebEditPro.

However, if you know HTML, you can view, insert or edit your Web page's HTML code.

This section explains

  • Viewing and Editing HTML
  • Inserting HTML
  • Cleaning HTML
  • Inserting Content from MS Office 2000
