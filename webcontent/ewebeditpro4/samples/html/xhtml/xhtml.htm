XHTML Sample

This sample shows how the editor can produce either HTML or XHTML. The format produced depends on the value of the publish attribute of the standard element in the config.xml file. The editor produces XHTML code when the content is saved and when viewing as HTML only if the publish attribute of the standard element and viewas element is set to "xhtml", respectively.

Editor 1 Produces Clean HTML

In config, the publish attribute is set to cleanhtml: <standard publish="cleanhtml">

Editor 2 Produces XHTML

In config, the publish attribute is set to xhtml: <standard publish="xhtml">
