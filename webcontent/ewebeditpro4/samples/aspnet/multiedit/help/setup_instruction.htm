ASP.Net Setup Instruction

The instruction applies to both the Multiedit Sample and the Multiple Config Sample.

 

  1. Open the Internet Service Manager. (Control Panel -> Administrative Tools -> Internet Service Manager)
  2. Navigate to the ewebeditpro4->samples->aspnet->multiedit directory
  3. Right click on the multiedit directory, select Properties, and then click the create button on the Directory tab(or Virtual Directory tab) of the Properties dialog. The Properties dialog for the mulutiedit directory is shown here.
  4. Click OK to update the mutiedit directory with new setting. The multiedit directory should now show the icon for an application root, as shown here.
